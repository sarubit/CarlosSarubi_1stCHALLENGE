# Como executar o projeto

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/sarubit/CarlosSarubi_1stCHALLENGE.git

# entrar na pasta do projeto first challenge
cd first-challenge

# instalar dependências
npm install

# executar o projeto
npm start
```
